# homebrew-clai

Homebrew tap for [clai](https://github.com/alexey1312/clai) - LLM-powered CLI help assistant.

## Installation

```bash
brew tap alexey1312/clai
brew install clai
```

## Update

```bash
brew upgrade clai
```

## Uninstall

```bash
brew uninstall clai
brew untap alexey1312/clai
```
